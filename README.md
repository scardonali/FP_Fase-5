## Descripción del Proyecto
Este repositorio contiene la solución automatizada en el lenguaje Python para el **Problema 5** del Banco de Problemas del curso. El software procesa y audita de manera estructurada las jornadas laborales semanales de un equipo técnico a través de arreglos matriciales bidimensionales y funciones de lógica condicional.

## Problema Seleccionado: Problema 5
* **Lógica de negocio:** Calcular la suma total de horas semanales trabajadas por persona (de Lunes a Viernes) a partir de una matriz de datos.
* **Clasificación:** * "Sobretiempo" si el acumulado excede el umbral estándar de 40 horas.
  * "Horario Estándar" si es igual o inferior al umbral.

## Requisitos de Ejecución
1. Contar con un intérprete de Python instalado.
2. Ejecutar el archivo principal desde la terminal de comandos: `python main.py`
